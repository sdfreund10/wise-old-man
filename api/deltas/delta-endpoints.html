Get Global Delta Leaderboards​

GET/deltas/leaderboard

Fetches the current top deltas leaderboard for a specific metric and period. Returns an array of DeltaLeaderboardEntry objects.


Query Params

FieldTypeRequiredDescription
periodPeriodtrueThe delta's time period.
metricMetrictrueThe delta's metric.
playerTypePlayerType?falseThe player's account type to filter by.
playerBuildPlayerBuild?falseThe player's account build to filter by.
countryCountry?falseThe player's country of origin to filter by.

Example Request

  • cURL
  • JavaScript / TypeScript
curl -X GET https://api.wiseoldman.net/v2/deltas/leaderboard?metric=agility&period=week&playerType=hardcore \
  -H "Content-Type: application/json"
const { WOMClient, Metric, Period, PlayerType } = require('@wise-old-man/utils');

const client = new WOMClient();

const deltasLeaderboard = await client.deltas.getDeltaLeaderboard({
  period: Period.WEEK,
  metric: Metric.AGILITY,
  playerType: PlayerType.HARDCORE
});
